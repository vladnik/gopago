== Test assignment from GoPago

1. Be able to list/add/edit/destroy book
2. Be able to check in/check out the book
3. Validate if it's available and when it was checked in/out
4. Write RSpec tests to cover functionality
5. Have a seed.rb ready to pre-populate some test data
